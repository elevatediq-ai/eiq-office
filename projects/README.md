# Project Boards (repo-backed markdown)

These markdown files act as canonical project boards in-repo when GitHub project API access is not available.

- Use them to move cards between columns by editing the file and committing the change.
- Each epic is linked to its issue URL so that clicking opens the issue.
- I have added comments on existing epic issues that point to their pillar file for easy reference.
